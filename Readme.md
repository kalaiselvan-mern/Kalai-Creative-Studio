# 🛒 Kalai Creative Studio - E-Commerce & Portfolio Platform

Kalai Creative Studio is a dynamic, full-stack e-commerce and portfolio web application built to showcase creative work and facilitate seamless online product purchases.

### 🌟 Key Features
- **Custom CMS:** Integrated a robust Content Management System for easy product and portfolio updates.
- **Secure Payments:** Seamless and secure checkout experience powered by Stripe payment gateway.
- **Optimized Media Storage:** Utilized Cloudinary for fast, optimized, and scalable product image storage.
- **State Management:** Efficient cart and user state handling using Zustand.
- **Responsive Design:** Fully responsive and visually appealing user interfaces crafted with Tailwind CSS.

### 💻 Tech Stack
- **Frontend:** React.js, Tailwind CSS, Zustand, Vite
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Third-Party Integrations:** Stripe (Payments), Cloudinary (Image Storage)
